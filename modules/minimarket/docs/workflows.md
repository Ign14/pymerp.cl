# Forma de trabajo esperada

## Operacion diaria
- Abrir POS y revisar dashboard.
- Revisar productos con stock critico.
- Ingresar compras temprano.
- Atender ventas locales con POS.
- Confirmar pedidos web durante el dia.

## Auditoria y control
- Cada movimiento queda registrado.
- Ajustes solo con motivo claro.
- Revisar ajustes recientes al cierre.

## Resolucion de errores
- Si el stock queda inconsistente, registrar un ajuste (no modificar stock directo).
- Si hubo venta sin internet, reintentar sincronizacion desde POS.
